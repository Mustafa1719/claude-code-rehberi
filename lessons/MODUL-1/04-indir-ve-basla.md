## 1. Kurs Dosyalarını İndir

Bu web sayfasındaki indirme butonuna tıkla — `claude-code-rehberi.zip` bilgisayarına inecek.

> **İndirme başarısız olursa:** [GitHub'dan manuel indir](https://github.com/Mustafa1719/claude-code-rehberi/releases/latest)

## 2. ZIP'i Çıkart

ZIP'e sağ tıkla → "Tümünü Çıkar" (Windows) veya çift tıkla (Mac). **Belgeler** klasörüne çıkart.

---

DÜŞÜN: Klasörü açtın mı? İçinde dosyalar var mı?

CEVAPLA: Evet / Hayır / Boş görünüyor

---

## 3. Claude Code'u Klasörden Başlat

Terminaline şunu yaz:

```
cd ~/Documents/claude-code-rehberi
claude
```

> Alternatif: `claude-code-rehberi` klasörüne sağ tıkla → "Yeni Terminal'de Aç" → `claude` yaz.

> ⚠️ **Önemli:** Claude yalnızca başlatıldığı klasördeki dosyaları görür. Her zaman `claude-code-rehberi` klasörünün içinden başlat.

---

DÜŞÜN: `>` işaretini gördün mü? Claude hazır mı?

CEVAPLA: Evet / Hayır

---

## 4. İlk Derse Başla

```
/baslat-0-0
```

Claude ders dosyasını okur, eğitmen moduna girer ve seni karşılar.

Kaldığın yerden devam etmek için: `/devam`
